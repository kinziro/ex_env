実験用自作Gym環境パッケージです

## Description

## Install
### インストール
```python
$ # aspacをダウンロードしたい階層で以下を実行
$ pip3 install -e git+https://github.com/kinziro/ex_env.git#egg=ex_env

$ # 以下を実行して'aspac'がでれば成功
$ pip3 list | grep myenv
```

### アップデート
```python
$ pip3 install -U myenv
```

### アンインストール
```python
$ pip3 uninstall myenv
```
